# 📚 玄空飞星

> 玄空飞星类书籍目录

## 简介

玄空飞星是清代蒋大鸿创的风水流派，以九星飞临位置判断吉凶，强调"山向配合"与"时运轮转"。是现代风水的主流流派之一。
