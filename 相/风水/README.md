# 📚 风水

> 风水类书籍目录

## 简介

风水又称堪舆术，是选择居住、葬埋地点的学问。分为阳宅（人居）和阴宅（墓地）两大类。核心在于"气"的聚集与流动，通过山川地形判断吉凶。
