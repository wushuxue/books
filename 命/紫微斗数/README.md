# 📚 紫微斗数

> 紫微斗数类书籍目录

## 简介

紫微斗数是宋代陈抟所创的命理学派，与八字并称命理两大支柱。以十四主星为主体，配合辅佐诸星和四化来分析命运。在港台地区影响力极大。
