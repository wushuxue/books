# 📚 八字

> 四柱八字类书籍目录

## 简介

八字命理源于唐代李虚中，经宋代徐子平完善，形成子平术体系。以出生年月日时四柱八字，配合五行生克制化来分析命运，是命理两大支柱之一（另一为紫微斗数）。
