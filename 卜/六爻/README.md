## 新增书籍简介

### 易冒
> 清 · 养西  著

《易冒》是清代重要的六爻著作，以简洁明了著称。书中内容涵盖起卦、断卦的核心方法，特别适合初学六爻者入门学习。

### 阐奥截要
> 清 · 胡仲一  著

《阐奥截要》旨在阐发六爻预测的深层奥义，截取精要，是六爻进阶学习的重要参考。
