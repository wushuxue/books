# 📚 卜 - 占卜

> 占卜类书籍目录

## 简介

"卜"是五术之一，包括六爻、梅花易数、奇门遁甲、大六壬、太乙神数、周易、河洛理数、择日等预测术数。合称"三式"的是太乙、奇门、大六壬。
