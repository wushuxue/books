# 📚 奇门遁甲

> 奇门遁甲类书籍目录

## 简介

奇门遁甲是三式之一，以"局"为单位，研究时空配合。被称为"帝王之学"，用于军事、政治、商业等重大决策。分为阳遁九局和阴遁九局，共十八局。
